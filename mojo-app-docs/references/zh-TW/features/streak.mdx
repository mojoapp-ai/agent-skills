# 每日連勝目標（Streak）

Streak 幫你追蹤每天是否達成健康目標，鼓勵你維持連續紀錄。就像遊戲中的每日任務，完成後累積連勝天數，幫助你把健康習慣變成日常。

## 可追蹤的項目

你可以自由選擇要追蹤哪些項目：

- **熱量目標** — 每日攝取熱量是否在目標範圍內（可設容錯範圍：±10% / ±15% / ±20%）
- **蛋白質目標** — 每日蛋白質攝取是否達標
- **運動** — 可選：任一運動 / 重訓 / 有氧
- **睡眠時數** — 預設 7 小時，可自訂
- **步數** — 預設 5,000 步，可自訂

## 達成模式

根據你的目標強度，選擇不同的達成模式：

| 模式 | 說明 | 適合誰 |
|------|------|--------|
| **任一項** | 任一項達成就算過關 | 剛開始養成習慣 |
| **至少 N 項** | 自訂至少要達成幾項 | 想逐步提高標準 |
| **全部達成** | 所有選中的項目都要達成 | 對自己有高要求 |

> 建議先從「任一項」開始，等習慣穩定後再提高難度。

## Streak 統計

- **當前連續天數** — 你目前已經連續達成幾天
- **最長連續天數** — 歷史最佳紀錄
- **本週達成天數** — 這週的表現
- **本月達成天數** — 這個月的表現

## Streak Freeze

每個月有一定次數的「免死金牌」。如果某天沒有達成目標，可以消耗一次 Freeze，連勝天數不會歸零。

這個設計是為了避免因為一天的中斷而失去動力——偶爾休息一天也是健康的一部分。
