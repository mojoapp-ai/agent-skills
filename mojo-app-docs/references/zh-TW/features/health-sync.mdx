# 健康資料同步

mojo 透過 Apple Health（iOS）和 Health Connect（Android）自動同步你的健康資料，省去手動輸入的麻煩。

## 支援的資料類型

| 資料類型 | iOS（Apple Health） | Android（Health Connect） |
|---------|-------------------|--------------------------|
| 體重 | ✅ | ✅ |
| 體脂百分比 | ✅ | ✅ |
| 骨骼肌 | ❌ | ❌ |
| 腰圍 | ✅ | ❌ |
| 營養（熱量/蛋白質/碳水/脂肪） | ✅ | ✅（每日彙總） |
| 活動熱量 | ✅ | ✅ |
| 靜息代謝 | ✅ | ✅ |
| 步數 | ✅ | ✅ |
| 運動記錄 | ✅ | ✅ |
| 睡眠 | ✅ | ✅ |
| 喝水 | ✅ | ✅ |
| 經期 | ✅ | ✅ |

> 骨骼肌目前 Apple Health 跟 Health Connect 都沒有支援，必須手動輸入。

## 如何授權

### iOS

1. 開啟 mojo → 設定 → Apple Health 同步
2. 點擊授權，建議全部開啟
3. 資料會自動在背景同步

如果不小心拒絕了授權，到 iOS「健康」App → 右上角頭像 → 找到 mojo → 開啟所有讀取權限。

### Android

1. 開啟 mojo → 設定 → Health Connect 同步
2. 授權所有資料類型
3. 資料會自動在背景同步

## 同步機制

- 資料有變更時自動同步；也可以在設定中手動觸發同步
- 使用增量同步，只抓取新增或修改的資料
- 如果你手動編輯了同步來的資料，mojo 會保留你的修改，不會被下次同步覆蓋

## 推薦搭配的 app 與裝置

**飲食記錄**：
- **MyFitnessPal**（免費）— 食物資料庫最完整
- **Calory**（付費）— 介面美觀，支援 AI 辨識食物

**運動與睡眠**：
- **Apple Watch** — 自動追蹤運動與睡眠
- **Garmin** — 運動手錶，透過 Apple Health / Health Connect 同步
- **Oura Ring** — 專注睡眠追蹤

**身體測量**：
- **Omron HBF-702T** — 體脂機，藍牙同步
- **Inbody H30** — 專業體組成分析

> 只要第三方 app 或裝置有寫入 Apple Health / Health Connect，mojo 就能自動讀取。
