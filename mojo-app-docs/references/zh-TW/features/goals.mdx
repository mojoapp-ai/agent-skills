# 目標設定

mojo 根據你的身體數據和活動量，自動計算每日營養目標。

## 每日營養目標

mojo 會自動計算每日的熱量和巨量營養素目標：

**計算方式：**
1. 根據你的體重、身高、年齡、性別，用 Mifflin-St Jeor 公式計算基礎代謝率（BMR）
2. 加上活動消耗，得到每日總熱量消耗（TDEE）
3. 根據你設定的減重速率，扣除熱量赤字
4. 分配蛋白質、碳水、脂肪的比例

**重訓日 vs 一般日：**

mojo 會根據當天的運動記錄自動調整營養目標：

| 營養素 | 重訓日 / 高活動量日 | 一般日 |
|--------|-------------------|--------|
| 蛋白質 | 體重 × 2.0 g/kg | 依活動程度 1.2–1.8 g/kg |
| 碳水 | 佔總熱量 45%（最低 80g） | 佔總熱量 50%（最低 100g） |
| 脂肪 | 剩餘熱量（最低 0.6 g/kg） | 佔總熱量 30%（最低 0.5 g/kg） |

系統判斷重訓日的條件：當天有重量訓練，或活動熱量 ≥ 250 卡路里。

> 如果有連結 Apple Health / Health Connect，mojo 會使用你的實際活動數據計算，而不是估算值。

## 其他目標

- **飲水目標** — 設定每日飲水量（mL / oz），在設定中可透過 Stepper 調整
- **睡眠目標** — 預設 7 小時，可自訂
- **步數目標** — 預設 5,000 步，可自訂
