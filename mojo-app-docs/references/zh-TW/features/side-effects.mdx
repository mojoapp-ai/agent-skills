# 副作用記錄

mojo 讓你記錄用藥過程中的副作用，幫助追蹤症狀變化，回診時也能提供醫師參考。

## 支援的副作用類型

**腸胃道（7 種）：**
- 噁心、腹瀉、食慾不振、嘔吐、便秘、消化不良、腹痛

**其他（3 種）：**
- 疲倦、頭暈、頭痛

## 嚴重程度

每種副作用可以標記三種嚴重程度：
- **輕微** — 可以忍受，不影響日常活動
- **中等** — 有明顯不適感
- **嚴重** — 明顯影響日常生活

## 如何記錄

1. 進入副作用記錄頁面
2. 選擇日期（可切換到過去的日期補記）
3. 點擊每個副作用的嚴重程度按鈕（再次點擊可取消）
4. 可在備註欄記錄其他症狀或感受
5. 自動儲存，不需手動按保存

每天只有一筆記錄，重新編輯會覆蓋前一次的內容。

## 資料如何被使用

- 過去 13 天的副作用記錄會納入 AI 分析，讓建議更精準
- 在資料管理中可按月份查看所有記錄
- 副作用記錄包含在備份中，支援跨平台還原
