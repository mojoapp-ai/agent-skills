# 功能介紹

mojo 整合咗減重管理所需嘅所有功能：

- **[健康數據同步](./health-sync)** — 自動同步 Apple Health / Health Connect 數據
- **[身體測量記錄](./body-measurement)** — 體重、體脂、肌肉量、腰圍、內臟脂肪
- **[用藥記錄](./medication)** — 注射記錄、庫存管理、用藥提醒
- **[副作用記錄](./side-effects)** — 10 種常見副作用，標記嚴重程度
- **[資料匯入與匯出](./data-backup)** — 備份還原、CSV 匯入、跨平台兼容
- **[目標設定](./goals)** — 營養目標、飲水與睡眠目標
- **[每日連勝目標](./streak)** — Streak 連續達成，養成每日健康習慣
- **[AI 回饋與報告](./ai-reports)** — 即時回饋與多專家深度分析
- **[庫存與成本分析](./cost-analysis)** — 藥物庫存追蹤、用藥成本計算
- **[帳戶與方案](../account/register)** — 會員註冊、訂閱方案、推薦碼獎勵
