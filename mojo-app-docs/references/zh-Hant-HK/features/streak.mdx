# 每日連勝目標（Streak）

Streak 幫你追蹤每日是否達成健康目標，鼓勵你維持連續記錄。好似遊戲入面嘅每日任務咁，完成後累積連勝日數，幫助你將健康習慣變成日常。

## 可追蹤的項目

你可以自由揀選要追蹤邊啲項目：

- **熱量目標** — 每日攝取熱量是否喺目標範圍內（可設容錯範圍：±10% / ±15% / ±20%）
- **蛋白質目標** — 每日蛋白質攝取是否達標
- **運動** — 可選：任一運動 / 重訓 / 有氧
- **睡眠時數** — 預設 7 小時，可自訂
- **步數** — 預設 5,000 步，可自訂

## 達成模式

根據你的目標強度，揀選唔同嘅達成模式：

| 模式 | 說明 | 適合邊個 |
|------|------|--------|
| **任一項** | 任一項達成就算過關 | 啱啱開始養成習慣 |
| **至少 N 項** | 自訂至少要達成幾項 | 想逐步提高標準 |
| **全部達成** | 所有選中嘅項目都要達成 | 對自己有高要求 |

> 建議先從「任一項」開始，等習慣穩定後再提高難度。

## Streak 統計

- **當前連續日數** — 你目前已經連續達成幾日
- **最長連續日數** — 歷史最佳記錄
- **本週達成日數** — 呢個星期嘅表現
- **本月達成日數** — 呢個月嘅表現

## Streak Freeze

每個月有一定次數嘅「免死金牌」。如果某日冇達成目標，可以消耗一次 Freeze，連勝日數唔會歸零。

呢個設計係為咗避免因為一日嘅中斷而失去動力——偶爾休息一日都係健康嘅一部分。
