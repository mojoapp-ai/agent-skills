# 健康數據同步

mojo 透過 Apple Health（iOS）和 Health Connect（Android）自動同步你的健康數據，慳返手動輸入嘅麻煩。

## 支援的數據類型

| 數據類型 | iOS（Apple Health） | Android（Health Connect） |
|---------|-------------------|--------------------------|
| 體重 | ✅ | ✅ |
| 體脂百分比 | ✅ | ✅ |
| 骨骼肌 | ❌ | ❌ |
| 腰圍 | ✅ | ❌ |
| 營養（熱量/蛋白質/碳水/脂肪） | ✅ | ✅（每日彙總） |
| 活動熱量 | ✅ | ✅ |
| 靜息代謝 | ✅ | ✅ |
| 步數 | ✅ | ✅ |
| 運動記錄 | ✅ | ✅ |
| 睡眠 | ✅ | ✅ |
| 飲水 | ✅ | ✅ |
| 經期 | ✅ | ✅ |

> 骨骼肌目前 Apple Health 同 Health Connect 都冇支援，必須手動輸入。

## 點樣授權

### iOS

1. 打開 mojo → 設定 → Apple Health 同步
2. 點擊授權，建議全部開啟
3. 數據會自動喺後台同步

如果唔小心拒絕咗授權，到 iOS「健康」App → 右上角頭像 → 搵到 mojo → 開啟所有讀取權限。

### Android

1. 打開 mojo → 設定 → Health Connect 同步
2. 授權所有數據類型
3. 數據會自動喺後台同步

## 同步機制

- 數據有變更時自動同步；亦可以喺設定中手動觸發同步
- 使用增量同步，只獲取新增或修改嘅數據
- 如果你手動編輯咗同步來嘅數據，mojo 會保留你的修改，唔會被下次同步覆蓋

## 推薦搭配的 app 與裝置

**飲食記錄**：
- **MyFitnessPal**（免費）— 食物資料庫最完整
- **Calory**（付費）— 介面靚，支援 AI 識別食物

**運動與睡眠**：
- **Apple Watch** — 自動追蹤運動與睡眠
- **Garmin** — 運動手錶，透過 Apple Health / Health Connect 同步
- **Oura Ring** — 專注睡眠追蹤

**身體測量**：
- **Omron HBF-702T** — 體脂磅，藍牙同步
- **Inbody H30** — 專業體組成分析

> 只要第三方 app 或裝置有寫入 Apple Health / Health Connect，mojo 就能自動讀取。
