# 升級為 mojo 好友

mojo 提供基本功能免費使用。升級為「mojo 好友」後，可以解鎖完整嘅 AI 分析與進階功能。

## 方案選擇

| 方案 | 說明 |
|------|------|
| **季付（三個月）** | 每三個月自動續訂 |
| **年付（一年）** | 最抵嘅選擇 |
| **早鳥優惠（一年）** | 新用戶限定，首次打開訂閱頁面後 21 日內有效 |

自動訂閱，隨時可取消。

## 免費 vs mojo 好友

| 功能 | 免費 | mojo 好友 |

| 即時 AI 回饋 | 每日一次 | 即時無限次 |
| 完整 AI 報告 | 有限次數 | 無限次 |
| 健康數據同步 | 有限次數 | 無限次 |
| 飲食記錄管理 | ✅ | ✅ |
| 藥物費用計算 | — | ✅ |
| 數據匯出與備份 | — | ✅ |
| 複製數據畀 AI 助手 | — | ✅ |
| Streak 目標自訂 | — | ✅ |
| 進階指標追蹤 | — | ✅ |
| 優先支援 | — | ✅ |

## 點樣升級

1. 到設定 → 訂閱與帳戶 → 升級訂閱
2. 揀選方案（季付、年付或早鳥）
3. 使用 Apple Pay（iOS）或 Google Pay（Android）付款
4. 即時生效

### 透過官方連結升級

我哋有時會分享官方連結 [mojoapp.ai/subscribe](https://mojoapp.ai/subscribe)（例如喺電子報入面）。

- 如果你已經安裝咗 mojo，撳個連結會直接喺 app 入面打開訂閱頁面。
- 如果你仲未安裝 app，個連結會帶你嚟到本頁，先了解各個方案。

## 取消訂閱

**iOS**：到 iPhone 設定 → Apple ID → 訂閱項目 → 搵到 mojo → 取消訂閱

**Android**：到 Google Play → 帳戶 → 付款與訂閱 → 訂閱項目 → 搵到 mojo → 取消訂閱

取消後，已付費期間仍可使用，到期後降為免費方案。

## 禮物訂閱

你亦可以購買禮物訂閱送畀朋友：

1. 到設定 → 訂閱與帳戶 → 購買禮物訂閱
2. 完成付款後，獲得一組 8 位數兌換碼
3. 將兌換碼發送畀對方

對方到設定 → 訂閱與帳戶 → 兌換禮物，輸入兌換碼即可啟用。如果對方已有訂閱，會自動延長期限。
