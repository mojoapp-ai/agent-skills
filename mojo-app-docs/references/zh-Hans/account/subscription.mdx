# 升级为 mojo 好友

mojo 提供基本功能免费使用。升级为「mojo 好友」后，可以解锁完整的 AI 分析与高级功能。

## 方案选择

| 方案 | 说明 |
|------|------|
| **季付（三个月）** | 每三个月自动续订 |
| **年付（一年）** | 最划算的选择 |
| **早鸟优惠（一年）** | 新用户限定，首次打开订阅页面后 21 天内有效 |

自动订阅，随时可取消。

## 免费 vs mojo 好友

| 功能 | 免费 | mojo 好友 |
|------|------|-----------|
| 即时 AI 反馈 | 每日一次 | 即时无限次 |
| 完整 AI 报告 | 有限次数 | 无限次 |
| 健康数据同步 | 有限次数 | 无限次 |
| 饮食记录管理 | ✅ | ✅ |
| 药物费用计算 | — | ✅ |
| 数据导出与备份 | — | ✅ |
| 复制数据给 AI 助手 | — | ✅ |
| Streak 目标自定义 | — | ✅ |
| 进阶指标追踪 | — | ✅ |
| 优先支持 | — | ✅ |

## 如何升级

1. 到设置 → 订阅与账户 → 升级订阅
2. 选择方案（季付、年付或早鸟）
3. 使用 Apple Pay（iOS）或 Google Pay（Android）付款
4. 立即生效

### 通过官方链接升级

我们有时会分享官方链接 [mojoapp.ai/subscribe](https://mojoapp.ai/subscribe)（例如在电子报中）。

- 如果你已经安装 mojo，点击链接会直接在 app 中打开订阅页面。
- 如果你还没安装 app，链接会带你来到本页，先了解各个方案。

## 取消订阅

**iOS**：到 iPhone 设置 → Apple ID → 订阅项目 → 找到 mojo → 取消订阅

**Android**：到 Google Play → 账户 → 付款与订阅 → 订阅项目 → 找到 mojo → 取消订阅

取消后，已付费期间仍可使用，到期后降为免费方案。

## 礼物订阅

你也可以购买礼物订阅送给朋友：

1. 到设置 → 订阅与账户 → 购买礼物订阅
2. 完成付款后，获得一组 8 位数兑换码
3. 将兑换码发送给对方

对方到设置 → 订阅与账户 → 兑换礼物，输入兑换码即可启用。如果对方已有订阅，会自动延长期限。
