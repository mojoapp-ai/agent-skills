# 身体测量记录

mojo 追踪以下身体数据，帮助你掌握减重过程中的身体变化。

## 追踪项目

**主要测量**：
- **体重**（kg / lb）
- **体脂百分比**（%）
- **体脂重量**（kg / lb）— 可独立记录或由体重 × 体脂率自动计算
- **肌肉量**（kg / lb）— 骨骼肌重量

**专项测量（付费版）**：
- **内脏脂肪等级**（1–59）
- **腰围**（cm / in）

## 内脏脂肪分级说明

mojo 将你输入的内脏脂肪等级（1–59）分为三个级距：

| 等级 | 范围 |
|------|------|
| 正常 | 1–9 |
| 轻度偏高 | 10–14 |
| 过高 | 15–59 |

> 内脏脂肪等级为体脂秤（如 TANITA、Omron 等）测量的指标，不同品牌可能采用不同的分级方式，mojo 采用以上分级供你参考。

## 输入方式

### 自动同步（建议）

使用支持 Apple Health / Health Connect 的智能体重秤，站上去测量后数据自动同步到 mojo。

目前可自动同步的项目：
- 体重、体脂百分比（iOS + Android）
- 腰围（仅 iOS，通过 Apple Health）

### 手动输入

到设置 → 数据管理 → 身体测量，点击右上角「+」按钮手动新增。

以下项目目前只能手动输入：
- 肌肉量（Apple Health 和 Health Connect 不支持骨骼肌重量）
- 内脏脂肪等级
- 腰围（Android）

## 进度图表

在「进度」Tab 可以查看趋势图表：
- 体重、体脂、肌肉量的变化趋势
- 支持切换时间范围：1 个月 / 3 个月 / 6 个月 / 全部
- 点击数据点可查看详细数值
- 自动根据你选择的单位（kg/lb）显示

图表以注射日为 X 轴基准，会获取注射前后 3 天内的测量数据。养成「注射前量一次体重」的习惯，图表会更完整。
