# 副作用记录

mojo 让你记录用药过程中的副作用，帮助追踪症状变化，复诊时也能提供医生参考。

## 支持的副作用类型

**肠胃道（7 种）：**
- 恶心、腹泻、食欲不振、呕吐、便秘、消化不良、腹痛

**其他（3 种）：**
- 疲倦、头晕、头痛

## 严重程度

每种副作用可以标记三种严重程度：
- **轻微** — 可以忍受，不影响日常活动
- **中等** — 有明显不适感
- **严重** — 明显影响日常生活

## 如何记录

1. 进入副作用记录页面
2. 选择日期（可切换到过去的日期补记）
3. 点击每个副作用的严重程度按钮（再次点击可取消）
4. 可在备注栏记录其他症状或感受
5. 自动保存，不需手动按保存

每天只有一笔记录，重新编辑会覆盖前一次的内容。

## 数据如何被使用

- 过去 13 天的副作用记录会纳入 AI 分析，让建议更精准
- 在数据管理中可按月份查看所有记录
- 副作用记录包含在备份中，支持跨平台还原
