# 每日连胜目标（Streak）

Streak 帮你追踪每天是否达成健康目标，鼓励你维持连续记录。就像游戏中的每日任务，完成后累积连胜天数，帮助你把健康习惯变成日常。

## 可追踪的项目

你可以自由选择要追踪哪些项目：

- **热量目标** — 每日摄取热量是否在目标范围内（可设容错范围：±10% / ±15% / ±20%）
- **蛋白质目标** — 每日蛋白质摄取是否达标
- **运动** — 可选：任一运动 / 重训 / 有氧
- **睡眠时数** — 默认 7 小时，可自定义
- **步数** — 默认 5,000 步，可自定义

## 达成模式

根据你的目标强度，选择不同的达成模式：

| 模式 | 说明 | 适合谁 |
|------|------|--------|
| **任一项** | 任一项达成就算过关 | 刚开始养成习惯 |
| **至少 N 项** | 自定义至少要达成几项 | 想逐步提高标准 |
| **全部达成** | 所有选中的项目都要达成 | 对自己有高要求 |

> 建议先从「任一项」开始，等习惯稳定后再提高难度。

## Streak 统计

- **当前连续天数** — 你目前已经连续达成几天
- **最长连续天数** — 历史最佳记录
- **本周达成天数** — 这周的表现
- **本月达成天数** — 这个月的表现

## Streak Freeze

每个月有一定次数的「免死金牌」。如果某天没有达成目标，可以消耗一次 Freeze，连胜天数不会归零。

这个设计是为了避免因为一天的中断而失去动力——偶尔休息一天也是健康的一部分。
