# 功能介绍

mojo 整合了减重管理所需的所有功能：

- **[健康数据同步](./health-sync)** — 自动同步 Apple Health / Health Connect 数据
- **[身体测量记录](./body-measurement)** — 体重、体脂、肌肉量、腰围、内脏脂肪
- **[用药记录](./medication)** — 注射记录、库存管理、用药提醒
- **[副作用记录](./side-effects)** — 10 种常见副作用，标记严重程度
- **[数据导入与导出](./data-backup)** — 备份还原、CSV 导入、跨平台兼容
- **[目标设定](./goals)** — 营养目标、喝水与睡眠目标
- **[每日连胜目标](./streak)** — Streak 连续达成，养成每日健康习惯
- **[AI 反馈与报告](./ai-reports)** — 即时反馈与多专家深度分析
- **[库存与成本分析](./cost-analysis)** — 药物库存追踪、用药成本计算
- **[账号与方案](../account/register)** — 会员注册、订阅方案、推荐码奖励
