# 库存与成本分析

mojo 帮你追踪药物库存的使用状况，并计算减重过程中的用药成本。

## 药物库存

每次拿到新的药物时，可以新增一笔库存记录：

- **药物类型** — 注射笔或口服药
- **总剂量 / 总数量** — 这盒药的**总含量**（见下方说明）
- **购买金额** — 金额与币种（支持 TWD、USD、JPY、EUR、CNY、KRW、HKD、SGD 等 16 种以上货币）
- **开封日期** — 何时开始使用

### 「总剂量」怎么填？

| 药物类型 | 填写方式 | 示例 |
|---------|---------|------|
| 注射笔 | 整支笔的**总 mg 数** | 一支有 4 剂、每剂 5mg → 填 **20**（mg） |
| 口服药 | 整盒的**总颗数** | 一盒 30 颗 → 填 **30**（颗） |

> ⚠️ 注意：注射笔的总剂量是整支笔的 mg 总量，**不是**每次的注射剂量。例如 Mounjaro 5mg 笔有 4 剂，总剂量应填 20mg，不是 5mg。如果填成每次剂量，库存卡片上的进度条会很快就超过 100%，你可以到库存管理修改。

库存卡片会显示进度条和已使用比例。用完的库存可以封存。

### 在用药记录中选择库存

新增完库存后，还需要在**用药记录**把每一次的注射或服药**关联到对应的库存**，系统才会自动扣除用量并计算成本。

在新增或编辑用药记录的页面上，会看到一个「库存」区块：

- 列出目前**这个药物类型**所有未封存的库存，点一下即可选取（换药物类型时会自动重新载入）
- 如果还没有可用的库存，可以直接从这里点「新增库存」
- 下次新增用药记录时，会自动带入你上次选的那笔库存，不用每次重选
- 若当下不想关联，可以选「不选择库存」；但这笔记录**不会列入成本分析**，库存用量也不会扣除

> ℹ️ 库存选择属于 PRO 功能，非订阅用户会看到锁定状态。

## 成本分析

成本分析帮你了解减重过程中的用药花费。这是订阅用户的高级功能。

**计算方式：**
1. 从你设定的「基准日」开始，只计算基准日之后的用药记录
2. 筛选出有关联库存且有填写金额的记录
3. 统计每支库存的实际使用量，取「标示总剂量」与「实际使用量」的较大值作为有效总量
4. 根据每盒库存的「金额 ÷ 有效总量」算出单位成本
5. 针对注射药物：每次注射成本 = 剂量 × 单位成本
6. 针对口服药物：每次服药成本 = 一颗的成本
7. 将符合条件的用药记录成本加总，如遇多币种则自动换算成你的显示币种

> 第 3 步的设计确保了即使总剂量填得比实际使用量低，总花费也不会超过库存的购买金额。

**显示内容：**
- 起始体重 → 目前体重 → 总共瘦了多少
- 累计用药花费
- **每公斤成本** — 你瘦的每公斤花了多少钱

**多币种支持：**
如果你的库存使用不同币种（例如在不同国家购买），mojo 会自动换算成你选择的显示币种，使用即时汇率。

**常见疑问：**
- **金额跟实际花费对不上？** 请依序检查：（1）库存的购买金额字段是空的（2）库存的「总剂量」是否填错（应填整支笔的总 mg，不是单次剂量）（3）部分用药记录没有关联到库存。到「药物库存」确认每笔都有填金额和正确的总剂量，并到「用药记录」逐笔确认都有选到对应的库存。
- **基准日怎么选？** 系统会根据你设定的基准日选项来决定起算点，只计算该日之后的用药记录。你可以在设置中调整。

**数据不足时：**
- 没有基线体重 → 提示你需要先记录体重
- 还没瘦 → 显示目前的累计花费，等有体重下降后才计算每公斤成本
- 用药记录没有关联库存或没填金额 → 该笔不计入成本
- 库存购买金额填 0 → 视为免费试用，会正常计入（总花费为 0）
