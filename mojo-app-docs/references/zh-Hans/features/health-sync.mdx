# 健康数据同步

mojo 通过 Apple Health（iOS）和 Health Connect（Android）自动同步你的健康数据，省去手动输入的麻烦。

## 支持的数据类型

| 数据类型 | iOS（Apple Health） | Android（Health Connect） |
|---------|-------------------|--------------------------|
| 体重 | ✅ | ✅ |
| 体脂百分比 | ✅ | ✅ |
| 骨骼肌 | ❌ | ❌ |
| 腰围 | ✅ | ❌ |
| 营养（热量/蛋白质/碳水/脂肪） | ✅ | ✅（每日汇总） |
| 活动热量 | ✅ | ✅ |
| 静息代谢 | ✅ | ✅ |
| 步数 | ✅ | ✅ |
| 运动记录 | ✅ | ✅ |
| 睡眠 | ✅ | ✅ |
| 喝水 | ✅ | ✅ |
| 经期 | ✅ | ✅ |

> 骨骼肌目前 Apple Health 跟 Health Connect 都没有支持，必须手动输入。

## 如何授权

### iOS

1. 打开 mojo → 设置 → Apple Health 同步
2. 点击授权，建议全部开启
3. 数据会自动在后台同步

如果不小心拒绝了授权，到 iOS「健康」App → 右上角头像 → 找到 mojo → 开启所有读取权限。

### Android

1. 打开 mojo → 设置 → Health Connect 同步
2. 授权所有数据类型
3. 数据会自动在后台同步

## 同步机制

- 数据有变更时自动同步；也可以在设置中手动触发同步
- 使用增量同步，只获取新增或修改的数据
- 如果你手动编辑了同步来的数据，mojo 会保留你的修改，不会被下次同步覆盖

## 推荐搭配的 app 与设备

**饮食记录**：
- **MyFitnessPal**（免费）— 食物数据库最完整
- **Calory**（付费）— 界面美观，支持 AI 识别食物

**运动与睡眠**：
- **Apple Watch** — 自动追踪运动与睡眠
- **Garmin** — 运动手表，通过 Apple Health / Health Connect 同步
- **Oura Ring** — 专注睡眠追踪

**身体测量**：
- **Omron HBF-702T** — 体脂秤，蓝牙同步
- **Inbody H30** — 专业体组成分析

> 只要第三方 app 或设备有写入 Apple Health / Health Connect，mojo 就能自动读取。
