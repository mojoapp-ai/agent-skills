# 用药记录

mojo 完整记录你的 GLP-1 用药过程，包含注射记录、库存管理和提醒功能。

## 注射记录

每次注射或服药时，可以记录以下信息：

- **日期与时间**
- **药物名称** — 支持 Mounjaro、Ozempic、Wegovy、Zepbound、Saxenda、Rybelsus 等
- **剂量** — 每种药物有标准剂量选项（例如 Wegovy：0.25mg、0.5mg、1.0mg、1.7mg、2.4mg）
- **注射部位** — 大腿（左右外上、外中上）、腹部（左中右，上中下）、上臂（左右）
- **关联库存** — 可连接到药物库存，追踪用量
- **备注** — 自由记录

口服药物（如 Rybelsus）会自动隐藏注射部位字段。

### 如何新增

1. 到「今天」Tab → 注射进度区块
2. 点击新增注射记录
3. 填写日期、药物、剂量等信息
4. 点击「保存」

## 库存管理

追踪你的药物库存，掌握剩余用量。

每笔库存记录包含：
- **药物名称与类型**
- **总剂量**（mg）或总数量（颗）
- **已使用量** — 根据关联的注射记录自动计算
- **购买金额与币种** — 支持 TWD、USD、JPY、EUR 等 16 种以上货币
- **开封日期**
- **进度条** — 可视化显示剩余比例

用完的库存可以封存，不影响历史记录。

## 用药提醒

mojo 会根据你的用药频率自动提醒注射时间。

**提醒逻辑：**
- 根据上次注射日期 + 用药间隔天数，计算下次应注射日期
- 当天如果到了注射日且尚未记录，会发送通知提醒
- 如果当天已经记录注射，就不会再提醒

**提醒消息示例：**
- 注射药物（每周）：「今天要打 Mounjaro 哦！上次是 2.5mg，打在腹部（右上）」
- 口服药物（每日）：「记得服用 Rybelsus 哦！上次是 7mg」

**设置方式：**
到设置 → 通知，可以自定义提醒时间、开关提醒。

如果你已停用某种药物（频率设为 0），该药物的提醒会自动停止。
