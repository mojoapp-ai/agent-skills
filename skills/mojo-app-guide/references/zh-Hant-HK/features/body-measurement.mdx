# 身體測量記錄

mojo 追蹤以下身體數據，幫助你掌握減重過程中嘅身體變化。

## 追蹤項目

**主要測量**：
- **體重**（kg / lb）
- **體脂百分比**（%）
- **體脂重量**（kg / lb）— 可獨立記錄或由體重 × 體脂率自動計算
- **肌肉量**（kg / lb）— 骨骼肌重量

**專項測量（付費版）**：
- **內臟脂肪等級**（1–59）
- **腰圍**（cm / in）

## 內臟脂肪分級說明

mojo 將你輸入嘅內臟脂肪等級（1–59）分為三個級距：

| 等級 | 範圍 |
|------|------|
| 正常 | 1–9 |
| 輕度偏高 | 10–14 |
| 過高 | 15–59 |

> 內臟脂肪等級係體脂磅（如 TANITA、Omron 等）量度嘅指標，唔同品牌可能採用唔同嘅分級方式，mojo 採用以上分級供你參考。

## 輸入方式

### 自動同步（建議）

使用支援 Apple Health / Health Connect 嘅智能磅，企上去量度後數據自動同步到 mojo。

目前可自動同步嘅項目：
- 體重、體脂百分比（iOS + Android）
- 腰圍（僅 iOS，透過 Apple Health）

### 手動輸入

到設定 → 資料管理 → 身體測量，點擊右上角「+」掣手動新增。

以下項目目前只能手動輸入：
- 肌肉量（Apple Health 同 Health Connect 唔支援骨骼肌重量）
- 內臟脂肪等級
- 腰圍（Android）

## 進度圖表

喺「進度」Tab 可以查看趨勢圖表：
- 體重、體脂、肌肉量嘅變化趨勢
- 支援切換時間範圍：1 個月 / 3 個月 / 6 個月 / 全部
- 點擊數據點可查看詳細數值
- 自動根據你揀選嘅單位（kg/lb）顯示

圖表以注射日為 X 軸基準，會獲取注射前後 3 日內嘅測量數據。養成「注射前量一次體重」嘅習慣，圖表會更完整。
