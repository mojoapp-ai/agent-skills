# 用藥記錄

mojo 完整記錄你的 GLP-1 用藥過程，包含施打記錄、庫存管理和提醒功能。

## 施打記錄

每次施打或服藥時，可以記錄以下資訊：

- **日期與時間**
- **藥物名稱** — 支援 Mounjaro、Ozempic、Wegovy、Zepbound、Saxenda、Rybelsus 等
- **劑量** — 每種藥物有標準劑量選項（例如 Wegovy：0.25mg、0.5mg、1.0mg、1.7mg、2.4mg）
- **注射部位** — 大腿（左右外上、外中上）、腹部（左中右，上中下）、上臂（左右）
- **關聯庫存** — 可連結到藥物庫存，追蹤用量
- **備註** — 自由記錄

口服藥物（如 Rybelsus）會自動隱藏注射部位欄位。

### 如何新增

1. 到「今天」Tab → 施打進度區塊
2. 點擊新增施打記錄
3. 填寫日期、藥物、劑量等資訊
4. 點擊「儲存」

## 庫存管理

追蹤你的藥物庫存，掌握剩餘用量。

每筆庫存記錄包含：
- **藥物名稱與類型**
- **總劑量**（mg）或總數量（顆）
- **已使用量** — 根據關聯的施打記錄自動計算
- **購買金額與幣別** — 支援 TWD、USD、JPY、EUR 等 16 種以上貨幣
- **開封日期**
- **進度條** — 視覺化顯示剩餘比例

用完的庫存可以封存，不影響歷史記錄。

## 用藥提醒

mojo 會根據你的用藥頻率自動提醒施打時間。

**提醒邏輯：**
- 根據上次施打日期 + 用藥間隔天數，計算下次應施打日期
- 當天如果到了施打日且尚未記錄，會發送通知提醒
- 如果當天已經記錄施打，就不會再提醒

**提醒訊息範例：**
- 注射藥物（每週）：「今天要打 Mounjaro 哦！上次是 2.5mg，打在腹部（右上）」
- 口服藥物（每日）：「記得服用 Rybelsus 哦！上次是 7mg」

**設定方式：**
到設定 → 通知，可以自訂提醒時間、開關提醒。

如果你已停用某種藥物（頻率設為 0），該藥物的提醒會自動停止。
